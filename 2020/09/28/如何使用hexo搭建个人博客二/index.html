发表于 2020-09-28 | 分类于 hexo blog

在上一篇文章中我们已经将博客成功部署到GitHub，接下来，我们需要对博客进行完善。

1.选择主题

去hexo的网站选择自己喜欢的主题，https://hexo.io/themes/
可点击主题图片上的眼睛符号进行预览，只有一部分主题可以预览。然后点击主题的名称获得主题的下载地址
https://github.com/next-theme/hexo-theme-next

在blog目录下打开git，输入

上述代码 后半段为克隆网址 + 克隆后的本地位置，过程可能较慢，耐心等待。
下载完成后可以看到themes目录下多了一个next文件夹。

打开配置文件

修改主题为next，如果是其他主题进行相应的修改即可。

这个时候主题就修改成功了，可以输入 hexo -s 进行本地预览。

这里区分站点配置文件 和主题配置文件，站点配置文件位于根目录下，主题配置文件位于themes/next 目录下
在主题配置文件中修改scheme可以选择不同的next主题。

在主题配置文件中可以设置语言菜单等等，上面的网站中有详细的介绍。

另外在next目录下有一个markdown说明文档可参考。
