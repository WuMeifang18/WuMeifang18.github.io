发表于 2020-09-28 | 分类于 博客搭建

这是一篇介绍零基础搭建博客网站的文章，一是为了自己的搭建过程，防止将来忘记，二是为想要学习搭建个人博客的同学提供相关方法。

1. 前期准备工作

1.1 相关的参考链接(可跳过)

  1. http://iissnan.com/progit/ （了解即可，这个网址有较为全面的关于git的介绍）
  2. http://theme-next.iissnan.com/getting-started.html 这个网站提供了设置hexo主题的方法。
  3. https://www.appinn.com/markdown/ 这是介绍markdown书写格式的网站
  4. https://www.jianshu.com/p/950f8f13a36c hexo博客图片问题

1.2 准备工作

本文针对windows系统，在开始之前，你需要进行以下操作

1.2.1 注册一个github账号

去官网注册即可 https://github.com/

1.2.2 安装git

网址：https://git-scm.com/
选择相应的版本，需要外网(fq)，不然下载速度很慢。也可以通过其他渠道。
安装git可以参考这篇博客：https://www.cnblogs.com/ximiaomiao/p/7140456.html。

Git安装之后需要进行一些基本信息设置

  1. 设置用户名：git config – global user.name ‘你再github上注册的用户名’;
  2. 设置用户邮箱：git config – global user.email ‘注册时候的邮箱’;

1.2.3 使用SSH密钥连接Github

参考教程：https://www.xuanfengge.com/using-ssh-key-link-github-photo-tour.html
使用SSH Keys在本机和GitHub之间建立一个安全的连接。这样在后面更新博客内容时就不用每次都输入用户名和密码了。
